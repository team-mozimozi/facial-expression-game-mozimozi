# facial-expression-game-mozimozi
AI 응용 로봇 SW / 비전응용프로젝트 2팀 모지모지

## 💡 프로젝트 개요

감정에 맞는 표정 짓기 게임

## ✍️ 프로젝트 설명

YOLO를 사용해 얼굴 표정을 인식하고, 주어진 얼굴 표정(이모지)와 같은 표정을 최대한 빨리 짓는 게임을 구현합니다.

## 🎯 목표

1. YOLO 사용해서 얼굴인식
2. 얼굴에서 감정 인식
3. 인식한 얼굴의 특징과 비교할 얼굴(이모지)의 유사도 측정
4. GUI(PyQT)로 게임 만들기
